# Install required dependencies

```bash
npm install

or

npm install --force
```

## Run tests

```bash
npm test
```

## Run on Android Studio

```bash
ionic cap run android
```

## Run on IOS Simulator

```bash
ionic cap run ios
```

## Open on Android Studio

```bash
ionic cap open android
```

## Open on IOS XCODE

```bash
ionic cap open ios
```
